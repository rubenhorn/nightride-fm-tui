# Nightride.FM TUI
A less resource-intensive way to listen to my favorite synth-wave radio station.

## Installation
1. Install the dependencies (mpv, rust toolchain)
2. Clone the repository
3. Run `cargo install --path .`

## Usage
1. Run `nightride`
2. Press `p` to play/pause
3. Press `q` to quit
